# CentOS

{% hint style="info" %}
本页面将为您提供在CentOS上使用一键脚本的方法。
{% endhint %}

{% hint style="warning" %}
注意：使用Beta版脚本可能会不稳定或对系统造成损害，请谨慎使用。

请确保脚本适合您的系统，否则可能会导致无法使用或系统损坏。

由于安全性问题，本站的所有脚本均不支持CentOS6，请使用更安全的CentOS7。
{% endhint %}

首先，在脚本列表中找到“CentOS专用”列表，然后找到适合您系统版本的脚本。

请确保您处于一个可以存放文件的目录。

如果您不确定该目录是否能存放文件，请使用下列命令转到Home目录。

```text
cd
```

使用下列命令来下载脚本：

```text
wget 脚本的下载地址
```

然后，输入以下命令来执行脚本：

```text
chmod +x 脚本名称 && bash 脚本名称
```

至此，您应该已经成功的执行了脚本，并可以使用服务。
